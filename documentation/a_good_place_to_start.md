# Backend Documentation

This folder contains the documentation of the system's backend.

- The [api_contract.md](./api_contract.md) file has the API Contract detailing the endpoints of the rest API, their input and their output.

- The [how_to_upload_config_files.md](./how_to_upload_config_files.md) file has instructions on how to upload the configuration files with the reference array and eligible measures to S3.
- The [pipeline.md](./pipeline.md) file has a short description of the application's CI/CD pipeline.
